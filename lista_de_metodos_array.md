Aqui está uma lista dos principais métodos de **Array** no JavaScript:

### Métodos de **leitura**:

1. **`Array.prototype.length`** - Retorna o número de elementos no array.
2. **`Array.prototype.indexOf()`** - Retorna o primeiro índice de um elemento no array.
3. **`Array.prototype.includes()`** - Verifica se um elemento está presente no array.
4. **`Array.prototype.find()`** - Retorna o primeiro elemento que satisfaz a condição fornecida.
5. **`Array.prototype.findIndex()`** - Retorna o índice do primeiro elemento que satisfaz a condição fornecida.
6. **`Array.prototype.at()`** - Retorna o elemento em um índice específico, podendo ser negativo para contar a partir do final.
7. **`Array.prototype.slice()`** - Retorna uma cópia superficial de uma parte do array em um novo array.
8. **`Array.prototype.concat()`** - Junta dois ou mais arrays.
9. **`Array.prototype.join()`** - Junta todos os elementos de um array em uma string.
10. **`Array.prototype.toString()`** - Retorna uma string representando o array.

### Métodos de **alteração**:

1. **`Array.prototype.push()`** - Adiciona um ou mais elementos no final do array.
2. **`Array.prototype.pop()`** - Remove o último elemento do array.
3. **`Array.prototype.shift()`** - Remove o primeiro elemento do array.
4. **`Array.prototype.unshift()`** - Adiciona um ou mais elementos no início do array.
5. **`Array.prototype.splice()`** - Adiciona ou remove elementos de qualquer posição no array.
6. **`Array.prototype.fill()`** - Preenche todos os elementos de um array com um valor específico, de um índice inicial até um índice final.
7. **`Array.prototype.reverse()`** - Inverte os elementos do array.
8. **`Array.prototype.sort()`** - Ordena os elementos do array.
9. **`Array.prototype.copyWithin()`** - Copia uma parte do array para outra parte dentro do mesmo array.

### Métodos de **iteração**:

1. **`Array.prototype.forEach()`** - Executa uma função em cada item do array.
2. **`Array.prototype.map()`** - Cria um novo array com os resultados da função aplicada a cada item do array.
3. **`Array.prototype.filter()`** - Cria um novo array com todos os elementos que passam no teste fornecido.
4. **`Array.prototype.reduce()`** - Aplica uma função acumuladora em cada elemento do array, resultando em um único valor.
5. **`Array.prototype.reduceRight()`** - Aplica uma função acumuladora em cada elemento do array, começando da direita para a esquerda.
6. **`Array.prototype.some()`** - Verifica se pelo menos um elemento no array passa no teste fornecido.
7. **`Array.prototype.every()`** - Verifica se todos os elementos do array passam no teste fornecido.
8. **`Array.prototype.find()`** - Retorna o primeiro elemento que satisfaz a condição fornecida.
9. **`Array.prototype.findIndex()`** - Retorna o índice do primeiro elemento que satisfaz a condição fornecida.
10. **`Array.prototype.flat()`** - "Achata" um array multidimensional em um único array.
11. **`Array.prototype.flatMap()`** - Primeiramente mapeia os elementos com uma função e depois "achata" o resultado.

### Métodos de **execução**:

1. **`Array.prototype.sort()`** - Ordena os elementos no array, com base em uma função de comparação (se fornecida).
2. **`Array.prototype.concat()`** - Combina dois ou mais arrays em um novo array.

### Métodos de **testes**:

1. **`Array.prototype.some()`** - Verifica se pelo menos um dos elementos no array satisfaz a condição fornecida.
2. **`Array.prototype.every()`** - Verifica se todos os elementos do array satisfazem a condição fornecida.
